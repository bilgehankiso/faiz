# Basit Faiz Hesaplayıcı Uygulaması

Bu proje, ana para, faiz oranı ve vade süresi verildiğinde basit faiz hesaplaması yapan bir uygulamadır.

## Özellikler
- Basit faiz formülü: Faiz = (Ana Para × Oranı × Vade) / 100
- Toplam tutar hesaplaması
- Kullanıcı dostu arayüz
- Python GUI ve Bash Script desteği

## Kurulum

Python sürümü gerekli:
```bash
python3 src/faiz_hesaplayici.py
```

Bash betiği kullanımı:
```bash
bash simple-interest.sh
```

## Kullanım

1. Ana para miktarını girin (TL)
2. Yıllık faiz oranını girin (%)
3. Vade süresini girin (Yıl)
4. Hesapla butonuna tıklayın veya scripti çalıştırın
5. Sonuç olarak toplam faiz ve vade sonundaki toplam tutar gösterilir
